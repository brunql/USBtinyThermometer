Usb Tiny Thermometer
====================

Simple thermometer based on `ATtiny2313` MCU and `DS18B20` thermal sensor.   
Software USB interface from project [USBtiny](http://www.xs4all.nl/~dicks/avr/usbtiny/).

**Hardware**: `ATtiny2313`, `DS18B20`   
**Firmware**: `C`, `USBtiny`   
**Software**: `C++`, `Qt`
