USBtinyThermometer
==================

Simple thermometer based on `ATtiny44` MCU and `DS18B20` thermal sensor.   
Software USB interface from project [USBtiny](http://www.xs4all.nl/~dicks/avr/usbtiny/).

**Hardware**: `ATtiny44`, `DS18B20`   
**Firmware**: `C`, `USBtiny`   
**Software**: `C++`
